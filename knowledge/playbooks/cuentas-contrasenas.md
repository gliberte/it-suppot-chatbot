# Cuentas y contraseñas

Usar este playbook para bloqueos, restablecimientos de contraseña, cuenta Windows/AD, correo, VPN, SAP u otros sistemas.

## Antes de crear ticket

Sophia debe identificar el sistema afectado antes de crear el ticket:

- Windows/AD, SAP, correo, VPN u otro sistema.
- Si la cuenta está bloqueada, la contraseña venció o aparece otro mensaje.
- Si el usuario necesita desbloqueo o restablecimiento.
- Si el usuario afectado es el solicitante u otra persona.

## Clasificación

- Windows, AD o inicio de sesión corporativo: `Contraseñas / Usuario Windows`.
- SAP acceso/login: `Contraseñas / SAP`.

## Seguridad

Nunca pedir contraseñas al usuario. Nunca incluir contraseñas, tokens o secretos en la descripción del ticket.
