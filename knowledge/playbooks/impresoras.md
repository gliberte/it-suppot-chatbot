# Impresoras

Usar este playbook para problemas de impresión, impresoras Zebra, Honeywell, etiquetas, papel atascado, calibración o cola de impresión.

## Antes de crear ticket

Pedir solo los datos necesarios:

- Modelo o ubicación de la impresora.
- Si muestra error, luces, papel atascado o problema de etiquetas.
- Si falla para todos los usuarios o solo desde un equipo.
- Si ya se reinició la impresora o se revisó la cola de impresión.

## Clasificación

Categoría sugerida: `Impresoras`.
Subcategoría sugerida: `Honeywell`, salvo regla específica del catálogo.

## Descripción útil

Debe incluir impresora afectada, ubicación, síntoma, mensaje de error y alcance del impacto.
