# Monitor y pantalla externa

Usar este playbook cuando el usuario reporte monitor sin imagen, monitor que no enciende, líneas, rayas, parpadeo, pantalla externa dañada o display externo con falla.

## Antes de crear ticket

Si el reporte es breve o incompleto, Sophia debe pedir solo dos o tres datos:

- Si el monitor enciende o queda totalmente sin imagen.
- Si el usuario ya probó otro cable, otro puerto o reiniciar el equipo.
- Si aparecen líneas, parpadeo, manchas, rayas o algún mensaje en pantalla.
- Ubicación o puesto cuando sea útil para atención presencial.

## Clasificación

Categoría sugerida: `Computadoras`.
Subcategoría sugerida: `Monitor`.
Prioridad normal: `Media`, salvo que afecte una operación crítica, sala de ventas, despacho, producción o más de un usuario.

## Descripción útil

La descripción debe incluir síntoma, pruebas realizadas, tipo de conexión si se conoce y ubicación.
