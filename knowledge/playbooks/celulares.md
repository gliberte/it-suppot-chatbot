# Celulares corporativos

Usar este playbook para celulares corporativos dañados, pantallas rotas, problemas de encendido, línea/SIM, aplicaciones móviles o fallas físicas.

## Antes de crear ticket

Pedir datos mínimos:

- Modelo del celular si el usuario lo conoce.
- Tipo de falla: física, encendido, línea/SIM, aplicación o rendimiento.
- Si hubo golpe, humedad o mensaje de error.
- Si el equipo es corporativo y quién lo utiliza, cuando el solicitante reporta por otra persona.

## Clasificación

Categoría sugerida: `Teléfonos`.
Subcategoría sugerida: `Celulares`.

## Prioridad

Usar `Alta` si el celular bloquea una operación crítica o comunicación de personal clave en campo.
